# Kartik Chouhan Portfolio

A modern, responsive, and professional personal portfolio built with React, Vite, and Tailwind CSS.

This portfolio highlights profile, skills, projects, experience, education, and contact details with a clean UI and a persistent light/dark theme toggle.

## Highlights

- Professional multi-section portfolio layout
- Responsive design for mobile, tablet, and desktop
- Persistent dark mode (saved in local storage)
- Reusable UI styling system with custom CSS variables
- Project showcase with live links and source links
- Fast development and optimized production build with Vite

## Tech Stack

- React
- Vite
- React Router
- Tailwind CSS
- React Icons

## Portfolio Routes

- / (Home)
- /skills
- /experience
- /project
- /education
- /contact

## Getting Started

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Preview production build locally:

```bash
npm run preview
```

- Update personal and social links in component files.
- Update project cards in the projects section.
- Tune colors and theme variables in src/index.css.
